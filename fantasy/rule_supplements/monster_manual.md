# Monster Manual
This is a list of monster stat blocks and any associated lore. The stat blocks are a simplification of the stats. For instance, 2d6 may be listed instead of a D6 multi-attack.

## Stat Block Key
H = hearts
DMG = damage
(P) = skill proficiency = +D6 to attacks
(E) = skill expertise = +2d6 to attacks
AV = Armor Value

### Glossary

#### Recharge <target number>
Each following turn, roll a D6. If it meets or exceeds the target number, it is recharged and ready to use. It's best to announce these rolls to build tension.
